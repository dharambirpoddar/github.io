We present a compositional motion planning framework for multi-robot systems based on encoding to satisfiability modulo theories (SMT). We leverage the power of off-the-shelf satisfiability modulo theories (SMT) solver (Z3) to solve the system of constraints. Moreover, we show how an SMT solver can be used to synthesize an optimal trajectory for each robot, where optimality is defined with respect to the available motion primitives. Several compositional frameworks for robot motion planning have been proposed in the past - motion description language, the maneuver automata, the sequential composition of closed-loop behavior in the absence of noise, and in the presence of noise. The motion planning problem has also been reduced to constrained dynamical simulation, where the trajectories are computed iteratively by solving a set of constraints on the system. Recently, SMT solvers have been used in motion planning with rectangular obstacles and in synthesizing integrated task and motion plans from plan outlines. However, composing a set of motion primitives to synthesize trajectories for a group of robots using an SMT solver has not been attempted before.
